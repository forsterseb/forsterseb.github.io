Heart Rate Monitor

Github
Python HTML/CSS/JS

Using an arduino and a few electrode pads I read the beats per minute of the heart and display it to a webpage so it can be used in OBS. Main reason for this was that I didn't want to spend a few hundred Euros just to record my heart activity - if I could do it myself for a lot cheaper.
The final product is quite accurate and give a nice new dimension to playing horror games or watching movies.

Discord Bot

Github
Python Docker

Main functionality of my discord bot is playing music of youtube videos in an voice channel. But there are many more small features included for example:

Python

This is a small script which defines a python decorator function. The decorator measures the runtime of each line in the decorated function and prints the result to the console.

Snap Memory Downloader

Github
Python

You are a bit oldschool and just like to have your pictures stored on your own harddrive? Or you want to leave Snapchat but dont want to leave all your images behind?
With this script you can download and save all your saved memories fully automatically. The only thing you need is to request your data from Snapchat and you are ready to go.
